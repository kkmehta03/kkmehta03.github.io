AI Presentation Maker

Let the Pinegrow AI Assistant create simple HTML presentations - Just give it the content or the topic.

Getting started

Open index.html or create a new page. Make sure that presentation.css is included on the page.

Presentation commands

Select the body element and ask the AI Assistant to:

Create a presentation

The AI Assistant will ask you about the topic of the presentation. Describe the topic or paste the content and click on the Chat button.

Slide commands

Select a .slide element and ask the AI Assistant to:

Insert a new slide
Change the slide layout to...
Add a suitable image
Split the slide into more slides

Customize design

Use the Design panel to customize the design.

Edit templates

Edit slide templates on templates.html. Give each slide template a name in Properties panel so that you can reference it from AI commands.
